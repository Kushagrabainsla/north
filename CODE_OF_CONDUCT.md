# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good intent, and engage constructively. Harassment, personal attacks, and discriminatory language are not tolerated in any project space - issues, pull requests, discussions, or related forums.

## Enforcement

Report unacceptable behavior privately to the project maintainer via the email address associated with the GitHub account, or open a private security advisory in this repository. All reports will be reviewed and investigated promptly and fairly.

The maintainer reserves the right to remove comments, commits, or other contributions that violate this Code of Conduct, and to ban contributors whose behavior is harmful to the community.
